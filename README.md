## Todo App With React 

you Cab add your Day to Day Task in my App. 


## Features 

1. You Can Easily Add Your Task by Clicking Start
2. You Can Edit The Task Also.
3. You Can Delete The Task by clicking Delete Button
4. It Has Three Filter, Like All , which shows all the tasks or Like Completed, which shows all the completed tasks and Like Pending, which shows only pending tasks.
